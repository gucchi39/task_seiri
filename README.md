# task_seiri（業務整理）

タグチ刺繍ネーム有限会社の業務改善プロジェクトの成果物置き場。

## 業務棚卸しマトリックス（ミニアプリ）

`index.html` は、工場長との面談で使うファシリテーション用のミニアプリ。

- **縦軸：拘束時間**（手が空かない度）／**横軸：属人性**（自分でないとできない度）
- 各業務を点で表し、**ドラッグして配置**できる
- **業務の追加**・**名前の変更**・**色（性質）の変更**・**削除**が可能
- 色は3種類：灰色＝実作業・対応／アンバー＝判断／緑＝切り出してもいい
- 配置や変更内容は `localStorage` に自動保存される
- 「元のマトリクスに戻す」で、最初に設計した11業務の配置に復元できる

### 使い方の意図

「これを切り出せ」と指示するのではなく、工場長自身が各業務をどの象限に置くかを選び、
「切り出してもいい（緑）」を付けていった結果、左上（拘束大・属人性低）に候補が集まる、
という流れを本人主導で作るためのツール。狙いはDTF出力の切り出しだが、それも本人に選ばせる。
